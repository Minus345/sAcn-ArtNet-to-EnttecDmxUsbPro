# sAcn to EnttecDmxUsbPro

sAcn to EnttecDmxUsbPro phyton script

Tipp:
- `python -m DMXEnttecPro.utils`  
  find usb Port
- you have to be root

## Used librarys:

- `pip install sacn`
- `pip install DMXEnttecPro`
- `pip install pyserial`

## Parameters:

- **usbPort:**
    - linux: `/dev/ttyUSB0`
    - windows: `COM4`
- **sAcn Universe**
